# Feel The Sound Sensory Panel
Feel the Sound: Διαδραστικός Πίνακας Αισθητηριακής Πλήρωσης (Sensory Panel)

Πολλοί μαθητές σε Ε.Ε.Ε.ΕΚ. (ειδικά στο φάσμα του αυτισμού ή με αισθητηριακές διαταραχές) αναζητούν οπτικά και απτικά ερεθίσματα για να ηρεμήσουν ή να συγκεντρωθούν. Οι εμπορικές λύσεις (Snoezelen rooms) είναι εξαιρετικά ακριβές. Η λύση μας: Ένας οικονομικός, "έξυπνος" τοίχος που μετατρέπει τα ηχητικά ερεθίσματα του περιβάλλοντος ή την κίνηση των χεριών σε εντυπωσιακά χρωματικά μοτίβα, χρησιμοποιώντας Open Hardware και Open Software.
